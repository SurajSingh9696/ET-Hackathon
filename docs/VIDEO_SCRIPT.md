## VISUAL GUIDE FOR VIDEO PRODUCTION

### Key Scenes to Capture:

1. **Problem Illustration** (15 seconds)
   - Stock footage: Person stressed with bills/financial documents
   - Text overlays with statistics

2. **Solution Introduction** (10 seconds)
   - Logo animation
   - Tagline: "AI-Powered Financial Wellness"

3. **Live Demo** (1 minute 45 seconds)
   - High-quality screen recording
   - Cursor highlights with annotations
   - Smooth transitions between sections
   - Zoom-ins on key metrics

4. **Architecture Explanation** (35 seconds)
   - Animated architecture diagram
   - Icons appearing for each component
   - Data flow animation

5. **Impact Visualization** (40 seconds)
   - Animated charts showing growth
   - Statistics appearing dynamically
   - Map of India with user distribution

6. **Closing** (30 seconds)
   - Presenter on camera (professional background)
   - Call-to-action screen with links

### Audio Notes:

- Background music: Upbeat, modern, not overpowering (30% volume)
- Voiceover: Clear, enthusiastic but professional
- Sound effects: Subtle transitions, "ding" for key metrics
- Pace: Energetic but not rushed, clear enunciation

### On-Screen Text:

Use consistently throughout:
- **Bold** for emphasis
- Highlight key numbers in color
- Smooth animations for statistics
- Icons for each dimension/feature

### Color Scheme:

- Primary: Purple/Indigo (brand colors)
- Accents: Green (positive), Red (alerts), Yellow (warnings)
- Background: White/Light gray for clarity

### Recommended Tools:

- Screen recording: OBS Studio, Loom
- Video editing: DaVinci Resolve, Adobe Premiere
- Animation: After Effects for graphics
- Voiceover: Audacity for recording/editing
